## Packages
recharts | Dashboard analytics and data visualization charts
date-fns | Robust date formatting and manipulation for bookings/schedules
framer-motion | Smooth animations for page transitions and UI interactions
clsx | Utility for constructing className strings conditionally
tailwind-merge | Utility for merging Tailwind classes safely

## Notes
- Using Recharts for the analytics dashboard (bar charts, line charts).
- React Hook Form + Zod used for all data entry (Tours, Settings).
- Framer Motion used for subtle entry animations and dialog transitions.
- Authentication handled via Replit Auth (using `useAuth` hook provided).
- Sidebar navigation layout for the main admin interface.
